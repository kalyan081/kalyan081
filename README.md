### 📂 Featured Repositories

* **🤖 [Autonomous Object Sorting Robot](https://github.com/kalyan081/autonomous-sorting-robot)** *A complete pick-and-place simulator for a 5-DOF robotic arm in Gazebo using ROS2 Humble, MoveIt2 motion planning, YOLOv8 perception for object classification, and a Streamlit telemetry dashboard.*
    * **Core Files**: Check out the state machine in [`ros2_ws/src`](https://github.com/kalyan081/autonomous-sorting-robot/tree/main/ros2_ws/src), YOLOv8 classification endpoints in [`vision_service`](https://github.com/kalyan081/autonomous-sorting-robot/tree/main/vision_service), and telemetry in [`dashboard`](https://github.com/kalyan081/autonomous-sorting-robot/tree/main/dashboard).

* **📚 [AI Study Assistant](https://github.com/kalyan081/ai-study-assistant)** *A local RAG-powered learning assistant with custom document ingestion (PDF, Word, TXT), FAISS vector search, Ollama LLM queries, auto MCQ/flashcard generation, and study tracker visualization.*
    * **Core Files**: Browse the text extraction in [`document_processor.py`](https://github.com/kalyan081/ai-study-assistant/blob/main/document_processor.py), semantic FAISS queries in [`vector_store.py`](https://github.com/kalyan081/ai-study-assistant/blob/main/vector_store.py), and page routes in [`app.py`](https://github.com/kalyan081/ai-study-assistant/blob/main/app.py).

* **🐛 [AI Code Debugger](https://github.com/kalyan081/ai-code-debugger)** *A local syntax analyzer and LLM-powered code reviewer. Employs Python AST parsing to locate compilation bugs and pipes files to a local Ollama model to yield structured, severity-ranked code diagnostics.*
    * **Core Files**: Review code analyzer logic in [`ast_analyzer.py`](https://github.com/kalyan081/ai-code-debugger/blob/main/ast_analyzer.py) and Ollama pipelines in [`llm_reviewer.py`](https://github.com/kalyan081/ai-code-debugger/blob/main/llm_reviewer.py).
